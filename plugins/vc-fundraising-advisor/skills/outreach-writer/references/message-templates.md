# Message Templates

Reusable templates for the three-message LinkedIn outreach sequence, plus the outreach rules. Each message has a MAIN version and a FALLBACK version for when the platform cannot pull the first name. Always have both loaded so the fallback fires automatically instead of sending an empty `[FIRST_NAME]`.

Replace every placeholder before sending:

- `[FIRST_NAME]` — the investor's first name
- `[YOUR INDUSTRY]` — your sector
- `[ONE-LINE DESCRIPTION]` — one line on what you built
- `[TARGET BUYER]` — who you build for
- `[KEY VALUE PROPOSITION]` — the value you deliver them
- `[YOUR STARTUP]` — your company name
- `[WHAT YOU DO]` — what your product does, in a few words
- `[YOUR NAME]` — your name

These are starting structures. Tailor each message with one real, specific signal about the investor before sending.

## 1. The Opener (sent after connection accepted)

**MAIN:**

```
Hi [FIRST_NAME], how are you? I wanted to ask if your VC is currently
looking to invest in startups focused on [YOUR INDUSTRY]?

We've built [ONE-LINE DESCRIPTION] that gives [TARGET BUYER]
[KEY VALUE PROPOSITION].

We'd love to present it to you! Let me know, all the best. [YOUR NAME]
```

**FALLBACK (no first name):**

```
Hi, how are you? I wanted to ask if your VC is currently looking to
invest in startups focused on [YOUR INDUSTRY]?

We've built [ONE-LINE DESCRIPTION] that gives [TARGET BUYER]
[KEY VALUE PROPOSITION].

We'd love to present it to you! Let me know, all the best. [YOUR NAME]
```

## 2. The Follow-up (day 3, if no reply)

**MAIN:**

```
Hi [FIRST_NAME], just wanted to follow up on my message below in case
it got lost!

We'd love to share how [YOUR STARTUP] is [WHAT YOU DO] for [TARGET BUYER].
Given your focus on the space, we think it could be a great conversation.

Would you be open to a quick 15-minute intro call? Let me know,
all the best. [YOUR NAME]
```

**FALLBACK (no first name):**

```
Hi, just wanted to follow up on my message below in case it got lost!

We'd love to share how [YOUR STARTUP] is [WHAT YOU DO] for [TARGET BUYER].
Given your focus on the space, we think it could be a great conversation.

Would you be open to a quick 15-minute intro call? Let me know,

```

## 3. The Final Ping (day 7, if no reply)

**MAIN:**

```
[FIRST_NAME], just wanted to ping here one last time in case my previous
messages got lost. Thanks!
```

**FALLBACK (no first name):**

```
Just wanted to ping here one last time in case my previous messages got
lost. Thanks!
```

## Outreach Rules

- 6 to 10 lines per message.
- One idea per message.
- Proof early: a number, a logo, or traction.
- One clear CTA.
- No big claims without numbers.
- No vision essay.

Look like a founder with priorities, not someone begging for attention. The sequence stops the moment the investor is not interested.
