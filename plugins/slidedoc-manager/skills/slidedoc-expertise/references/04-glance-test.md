# The Glance Test -- Rapid Comprehension Evaluation

> **Source:** Nancy Duarte, HBR article "Do Your Slides Pass the Glance Test?" (2012)
> and Duarte's free Glance Test tool.

## What Is the Glance Test?

The **Glance Test** is a method developed by Nancy Duarte to evaluate whether a slide
or Slidedoc page can be understood quickly and clearly. It is based on the principle
that audiences should be able to comprehend each page's main message within
approximately **3 seconds**.

Duarte calls effective slides **"glance media"** -- they should function like billboards,
delivering meaning through a single, brief glance.

## How to Apply the Glance Test

### Step-by-Step Process

1. **Look at the page for exactly 3 seconds**, then look away
2. **Determine** whether you can remember and articulate the main message
3. **If you cannot** recall the core message, the page has too much information
   or poor visual hierarchy
4. **Return to the page** to pare it down or redesign it

### For Slidedocs Specifically

While Slidedocs are designed to be read (not glanced at during a live presentation),
the Glance Test still applies at a structural level:

- Each page should have a **single, clear main message** identifiable at a glance
- The reader should immediately understand **what the page is about** and **where to start reading**
- Supporting details can be dense, but the **headline/takeaway must be instant**
- The visual hierarchy must make the **most important point unmissable**

## What the Glance Test Evaluates

| Element | Pass | Fail |
|---------|------|------|
| **Focus** | Single, clear point per page | Multiple competing messages |
| **Hierarchy** | Key message is visually dominant | Everything looks equally important |
| **Contrast** | Important elements stand out | Flat, undifferentiated design |
| **Whitespace** | Clean, breathing room | Cluttered, cramped |
| **Text density** | Concise, scannable | Wall of text |
| **Visual clarity** | Charts/images reinforce the message | Decorative or confusing visuals |

## The Science Behind It

- People **cannot multitask** effectively: they can't read dense text and listen simultaneously
- **Visual information is processed faster** than textual information
- Memory research shows **visuals are retained longer** than text alone
- Cognitive load theory: **overloaded pages cause comprehension failure**

## Applying to Slidedoc Review

When reviewing a Slidedoc, apply the Glance Test to each page by asking:

1. Can I identify the **main message** in 3 seconds?
2. Do I know **where to start reading**?
3. Is the **visual hierarchy** clear (what's most vs. least important)?
4. Are the **visuals reinforcing** the text, not competing with it?
5. Would someone unfamiliar with the topic understand **the page's purpose**?

If any answer is "no," the page needs redesign.
